# UCF Computer Science I – Fall 2024 🚀

This repository contains my coursework, projects, and implementations from **COP 3502C – Computer Science I** at the **University of Central Florida (UCF)** during the **Fall 2024** semester with Dr. Tanvir Ahmed.
The course focuses on developing a strong foundation in **C programming**, along with essential **data structures** and **algorithms** that are fundamental to computer science.

---

## 📁 Contents

This repository includes:

- ✅ **Programming Assignments (PAs)**  
  Real-world style coding assignments applying key data structures and algorithms.

- 💻 **Lab Exercises**  
  Hands-on practice problems to reinforce weekly topics.

---

## 🧠 Topics Covered

### ✅ C Programming Review
- Pointers
- Structs
- Strings

### 📦 Dynamic Memory Allocation
- malloc, calloc, free
- Memory management in struct-based programs

### 🔢 Base Conversion
- Converting numbers between binary, decimal, and hexadecimal

### 🔗 Data Structures
- **Linked Lists** (Singly, Doubly)
- **Stacks** (Array-based & Linked)
- **Queues** (Linear, Circular)
- **Binary Trees**
- **Binary Search Trees (BST)**
- **Binary Heaps** (Min/Max)
- **AVL Trees**
- **Tries**
- **Hash Tables** (Open addressing and chaining)

### 📚 Algorithms
- **Recursion**
- **Summation**
- **Recurrence Relations**
- **Algorithm Analysis** – Big-O, Omega, Theta
- **Binary Search**
- **O(n²) Sorting Algorithms** – Bubble, Insertion, Selection Sort
- **Merge Sort**
- **Quick Sort**

### 🧮 Bit Manipulation
- Use of **bitwise operators** for efficient computation
